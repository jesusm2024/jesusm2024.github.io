OnSite

Overview:

OnSite is an app created for Poatek, an IT consulting firm based in Brazil that operates on a hybrid work model, to quickly plan their meetings based on where employees are working for the day.

Technologies:

  • Frontend: React
  • API: Integrated with Poatek's internal API to retrieve employee work status.

Features:

  • API integration to dynamically pull employee work locations.
  • Clean and simple UI for quick meeting scheduling.
