Computer Graphics

Overview:

These projects explore the creation of interactive 3D environments using WebGL, focusing on user-driven exploration, dynamic animations, and real-time manipulation of 3D objects. The work demonstrates expertise in implementing advanced graphics techniques, such as lighting and shading models, perspective and orthographic projections, and complex jointed assemblies. Through intuitive user interfaces with controls for camera navigation, lighting adjustments, and object manipulation, these projects emphasize the seamless integration of technical depth with engaging, hands-on user experiences in immersive 3D worlds.

Project A

Built dynamic 3D toy models of a bike and helicopter using WebGL, featuring animated jointed assemblies and interactive controls that offer a playful, nostalgic experience with smooth animations and user engagement.

Project B

Developed a large, interactive 3D world with a movable camera where users can explore side-by-side perspective and orthographic views, control jointed assemblies, and adjust settings through sliders and keyboard/mouse inputs

Project C

Created an interactive 3D environment using WebGL where users can adjust light source positions and compare Gouraud and Phong shading techniques, highlighting lighting effects on various materials in real-time.
