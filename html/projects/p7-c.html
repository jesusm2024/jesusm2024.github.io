Project C

Overview:

Build an interactive 3D environment that showcases various lighting models and shading techniques using WebGL. Users can explore this environment while dynamically adjusting the position of a light source and observing how different materials respond to lighting effects. The project features side-by-side comparisons of Gouraud and Phong shading techniques, enabling users to see the differences in rendering quality and realism. The interactive interface includes controls for managing lighting properties, material settings, and the overall scene dynamics.

Technologies:

  • WebGL: Utilized for rendering the 3D scene, creating geometries, and managing lighting calculations.
  • JavaScript: Handles user interactions, event listeners, and updates to the 3D scene based on user input.
  • HTML5 Canvas: Serves as the platform for displaying the interactive 3D environment in the browser.
  • Matrix Transformations (glMatrix.js): Employed for handling transformations and camera movements within the 3D space.
  • GLSL Shaders: Custom shaders for implementing lighting models, material properties, and shading techniques.

Features:

  • Interactive Light Source: Users can control the position of the light source in 3D space using sliders for the X, Y, and Z axes, allowing real-time adjustments and immediate visual feedback.
  • Shading Techniques: The project implements two shading techniques:
    • Gouraud Shading: Calculates vertex colors and interpolates them across polygons.
    • Phong Shading: Computes lighting at every pixel for a more detailed and realistic appearance.
  • Material Properties: Users can toggle between different material settings (e.g., matte, shiny) to see how they affect the appearance of objects under different lighting conditions.
  • Dynamic Scene Updates: The scene updates in real time based on user interactions, showcasing the effects of lighting and material changes immediately.
  • Side-by-Side Comparison: A dual viewport allows users to compare the results of Gouraud and Phong shading techniques simultaneously.

Demo:

Role:

  • 3D Environment Design: Developed the overall 3D scene, including the creation of various objects and the implementation of lighting models.
  • Lighting Model Implementation: Implemented both Gouraud and Phong shading techniques, ensuring that they were accurately represented in the rendering process.
  • Shader Development: Wrote GLSL shaders for lighting calculations, material properties, and shading effects, optimizing for performance and visual quality.

Challenges:

  • Lighting Calculation Complexity: Implementing accurate lighting calculations for both Gouraud and Phong shading posed a challenge. This was addressed by optimizing the GLSL shaders to ensure that they could efficiently handle the necessary computations without compromising performance.
  • Material Property Representation: Accurately representing different material properties in the shading calculations was complex. I solved this by establishing a clear mapping between user inputs and the material characteristics, allowing for intuitive adjustments.
