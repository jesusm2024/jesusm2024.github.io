Pulse

Overview:

Designed and developed a multifunctional alarm clock in collaboration with a peer. The clock was created to challenge the user to perform various tasks, such as reaching a target heart rate or solving trivia questions, in order to turn off the alarm. The project required integrating several hardware components, including an LCD screen, joystick, heartbeat sensor, and a Micro:bit microcontroller. The focus was on hardware-software integration, where custom driver code was written to interface with each sensor and component, ensuring seamless functionality. We also managed the budgeting, purchasing of materials, and adhered to a strict timeline for development and presentation.

Technologies:

  • Languages: C
  • Microcontroller: Micro NRF52833
  • Sensors: Joystick (SparkFun COM-15168), Heart Rate/Pulse Sensor (SparkFun SEN-11574)
  • Display: Adafruit LCD Screen (Product ID 1770)

Features:

  • Set Alarm with Joystick: Users could set a timer using the joystick, with confirmation via a click.
  • Randomized Alarm Challenges:
    • Heart Rate Task: Users had to increase their heart rate by 10% above their baseline within 30 seconds.
    • Arrow Sequence Memorization: Users had to input a series of arrows correctly using the joystick.
    • Math Trivia: Users solved a math problem by selecting the correct answer using the joystick.
    • General Trivia: Similar to math trivia but with general knowledge questions.
  • Failure Handling: If a user failed a challenge, the alarm would reset, requiring them to redo the task.
