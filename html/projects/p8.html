Generative Methods

Overview:

This project showcases a collection of creative, animated, and interactive artwork and tools built using p5.js. The portfolio includes various animated gifs, brushes, particle systems, adjustable objects, and masks, all created with the purpose of exploring creativity and expression through code.

Technologies:

  • Languages: HTML, JavaScript, CSS
  • Libraries: p5.js
  • Glitch: A platform used to host and manage the different projects.

Features:

  • Animated Gifs: Six different animated gifs showcasing abstract concepts like spinning circular patterns, futuristic cityscapes, and space nebulas
  • Brushes: Multiple drawing tools, including flower patterns, random triangles, and Bezier curve brushes. These tools allow continuous drawing with shapes, colors, and rotations.
  • Particle Systems: Four different particle systems that showcase varied behaviors like explosions, shooting stars, and particle interaction with user input.
  • Adjustable Objects: Three interactive spaces where users can create custom unicorns, robots, and rocket ships by adjusting parameters such as size, color, and speed.
  • Masks: Two mask designs with customizable features and responsive movement based on user input.

Role:

  • Sole Developer: Took ownership of the entire development process, from initial concept ideation through to coding, testing, and deployment of each interactive project in the portfolio.
  • Creative Design & Implementation: Designed and coded all interactive elements using p5.js, ensuring that each project—from animated gifs to particle systems—was both functional and visually engaging.
  • Debugging & Iteration: Identified and solved bugs related to rendering performance and unexpected behavior in particle systems and interactive elements.

Challenges:

  • Unexpected Particle System Behavior: Initially, none of the particle systems worked as expected, so you iterated on the design by experimenting with forces and interactions until the systems evolved into something you were satisfied with.
  • Customizable Parameters for Objects and Masks: Creating customizable spaces and masks required a balance between flexibility and usability. You implemented several controls to adjust parameters, allowing users to create custom versions of objects like unicorns or masks.
  • Secondary Motion with Masks: Designing responsive secondary motion in the masks (like face contour tracking and shadow following) posed a challenge, but you solved this by leveraging tracking data to add realistic movement.

Takeaways

  • Creativity through Code: Through this project, you discovered how code can be used as a medium for art. You learned how to combine basic elements like shapes, colors, and transformations to create complex visual effects and interactive experiences.
  • Modularity and Flexibility: One key takeaway is the power of modular design—by creating adjustable parameters, you allowed users to interact with and modify your artwork, adding an extra layer of engagement and creativity.
