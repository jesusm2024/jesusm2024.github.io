Study Buddies

Overview:

Study Buddies is a mobile app that connects Northwestern students with study partners by matching them based on their class schedules and study preferences. Students register with a class code, input their schedule, and browse potential matches. Built with React for the frontend and Firebase for backend functionality, it creates a simple, intuitive user experience.

Technologies:

  • Frontend: React
  • Backend: Firebase

Features:

  • Class and Schedule Matching: Matches students based on their registered classes and availability.
  • Swipe-Based Matching: A Tinder-like interface allows users to swipe through potential study partners.
  • Preferences: Customize matching by study preferences and other user details.

Role:

Worked in a team of seven, contributing to the development of key matching features and backend integration. Helped brainstorm and implement user flows while ensuring cross-team collaboration with pair programming.

Challenges:

  • Team Collaboration: Managing contributions across a large team required clear communication and Agile sprint planning. We overcame this with structured pair programming and regular team check-ins.
  • Bus Factor: Mitigated reliance on any single team member by cross-training and ensuring documentation for all features.

Takeaways

  • Prioritize building a Minimum Viable Product (MVP) to focus on core features.
  • Use Agile sprints to manage design iterations and keep the team aligned.
  • Foster collaboration with pair programming to distribute knowledge across the team and avoid over-reliance on any one member.
