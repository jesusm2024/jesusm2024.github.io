Avatar the Last Airbender Wiki

Overview:

Developed a website using HTML, JavaScript, CSS, and the Anime.js library to create a fan wiki for Avatar: The Last Airbender. The site includes a breakdown of characters, episodes, and seasons across three books (Water, Earth, Fire) and integrates APIs to display characters and trivia questions about the show. Additionally, users can explore comics, YouTube analyses, and official Avatar resources.

Technologies:

  • Languages: HTML, JavaScript, CSS
  • Libraries: Anime.js for animations
  • APIs: Last Airbender API for character data, Sample APIs for trivia questions

Features:

  • Season-by-season breakdown of episodes with synopses and character introductions for Avatar: The Last Airbender.
  • API integration to display character details from both Avatar and The Legend of Korra, with filtering options by nation.
  • Trivia game with questions pulled from an API, allowing users to test their knowledge of the show.
  • Subtle animations via Anime.js to enhance user experience without distracting from the content.

Role:

  • Created a modular system for dynamically loading characters, episodes, and trivia questions.

Challenges:

  • API Integration and Filtering: Faced challenges when integrating the Last Airbender API and building filters for character affiliation. Solved this by optimizing the API call structure to fetch data based on filters dynamically and updating the UI accordingly.
  • Trivia Game Logic: Implemented custom logic to validate user answers in the trivia game against the API-stored correct answers, ensuring real-time feedback for the user.
  • Animation Integration: Balanced the use of Anime.js for animations to enhance the look and feel of the website without overwhelming users or detracting from the content.
